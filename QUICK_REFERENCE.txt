================================================================================
📱 FIREBASE COLLECTIONS
================================================================================

USERS Collection
  Document ID: User's UID
  Fields: fullName, email, phone, dob, role, createdAt

DOCTORS Collection
  Document ID: Auto-generated
  Fields: name, specialization, languages, yearsExp, rate, rating, bio, image

APPOINTMENTS Collection
  Document ID: Auto-generated
  Fields: patientId, doctorId, sessionType, date, time, reason, emergency, status

================================================================================
🐛 TROUBLESHOOTING
================================================================================

Doctors Not Showing?
  → Run: seedDoctors() in console
  → Check Firebase Console → doctors collection

Language Not Changing?
  → Clear browser cache (Ctrl+Shift+Delete)
  → Refresh page (F5)
  → Check console for errors (F12)

Registration Not Working?
  → Check console for errors (F12)
  → Make sure email is unique
  → Password must be 6+ characters

Appointments Not Saving?
  → Make sure you're logged in
  → Check console for errors
  → Verify all form fields filled
